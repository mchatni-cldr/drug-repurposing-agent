1. Identify all drugs, proteins, diseases, pathways, and biomarkers mentioned
2. Extract relationships between entities
3. Use standardized entity names (e.g., "STAT3" not "Stat3" or "stat-3")

   - overexpressed_in, downregulated_in, associated_with

   - causes, prevents, reduces, increases
   - participates_in, regulates, modulates

   - Explicit statements = 0.9-1.0
   - Strong implications = 0.7-0.9
   - Weak associations = 0.5-0.7

6. Cite the source sentence for each triplet

FORMAT:
Return ONLY a JSON array with this structure (no preamble, no markdown):
[
  {
    "subject": "entity_name",
    "subject_type": "drug|protein|disease|pathway|biomarker",
    "predicate": "relationship_type",
    "object": "entity_name", 
    "object_type": "drug|protein|disease|pathway|biomarker",
    "confidence": 0.95,
    "source_sentence": "exact sentence from text"
  }
]
